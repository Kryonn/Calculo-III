# Cálculo III

Este repositório reúne exercícios resolvidos de Cálculo 3, provenientes das listas de exercícios da disciplina de Cálculo 3 oferecida pela Universidade de São Paulo (USP). O objetivo é servir como material de estudo e referência para estudantes de engenharia, matemática, física e áreas correlatas, oferecendo exemplos práticos e detalhados das principais técnicas e conceitos abordados na disciplina.

## Tópicos abordados

1. **Derivadas parciais**
   - Cálculo de derivadas e diferenciais
   - Gradiente e derivadas direcionais


2. **Integrais múltiplas**
   - Integrais duplas e triplas
   - Mudança de variáveis (polares, cilíndricas, esféricas)
   - Aplicações: áreas, volumes, centro de massa  

3. **Campos vetoriais**
   - Divergente, rotacional e linhas de campo
   - Integrais de linha e superfície  

4. **Teoremas de cálculo vetorial**
   - Teorema de Green  
   - Teorema de Stokes  
   - Teorema da divergência (Gauss)
